# Contributing to BaseTap

## Development Setup

```bash
forge install
forge build
forge test
```

## Testing

Run all tests:
```bash
forge test -vvv
```

Run with gas reporting:
```bash
forge test --gas-report
```

## Code Style

- Use Solidity 0.8.23
- Follow naming conventions
- Add natspec comments
- Write comprehensive tests

## Pull Requests

1. Fork the repository
2. Create feature branch
3. Write tests
4. Submit PR with description

## Security

Report vulnerabilities to security@basetap.xyz
